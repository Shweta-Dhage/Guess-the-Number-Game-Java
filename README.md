# Guess the Number Game | Java

This is a Java console game where the computer chooses a random number between 1 and 100. You try to guess it, and the game tells you if your guess is too high or too low. It also counts how many tries you took to guess it right.

## How to Run
1. Open `Main.java` in any Java IDE or run it from the terminal.
2. Follow the instructions on the screen to play.

## Features
- Random number from 1 to 100
- You enter guesses in the console
- Hints for high or low guesses
- Counts your attempts
